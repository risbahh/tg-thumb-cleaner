# TG Thumbnail Watermark Cleaner

A Telegram userbot that watches a source channel, removes text watermarks from video thumbnails, strips `@mentions` from captions, and forwards to your index channel — all without re-downloading the full video.

## How it works

```
New video posted in SOURCE_CHANNEL
        ↓
Download thumbnail (small JPEG, ~5–30 KB)
        ↓
EasyOCR detects all text in the thumbnail
        ↓
OpenCV inpaints (erases) the detected text regions
        ↓
Send to DEST_CHANNEL with cleaned thumbnail + cleaned caption
(video sent by file_id — no full re-download)
```

## Setup

### 1. Get Telegram API credentials
Go to [my.telegram.org](https://my.telegram.org) → API development tools → create app.

### 2. Generate SESSION_STRING
```bash
pip install pyrofork tgcrypto-pyrofork
python session_gen.py
```
Copy the output string.

### 3. Deploy to Railway
1. Fork this repo
2. Connect to Railway → New Project → Deploy from GitHub
3. Add Variables (see `.env.example`)
4. Deploy

### 4. Add the userbot to the source channel
The account whose `SESSION_STRING` you generated must be a **member** of `SOURCE_CHANNEL`.

## Environment Variables

| Variable | Required | Description |
|---|---|---|
| `API_ID` | ✅ | From my.telegram.org |
| `API_HASH` | ✅ | From my.telegram.org |
| `SESSION_STRING` | ✅ | Generated by session_gen.py |
| `SOURCE_CHANNEL` | ✅ | Channel to watch (`@username` or `-100xxx`) |
| `DEST_CHANNEL` | ✅ | Your index channel ID (negative integer) |
| `ADMINS` | Optional | Your Telegram user ID for commands |
| `DELAY` | Optional | Seconds between forwards (default: 2) |
| `LOG_CHANNEL` | Optional | Get startup notifications |

## Commands (DM the userbot)

| Command | Description |
|---|---|
| `/status` | Session stats — forwarded, thumbnails cleaned, failed |
| `/help` | Help message |

## Notes

- **EasyOCR model** (~100 MB) downloads automatically on first start
- Works on **CPU only** — no GPU needed — runs fine on Railway free tier
- The full video is **never downloaded** — only the small thumbnail JPEG
- Telegram thumbnail limit: JPEG, max 200 KB, max 320×320 px
